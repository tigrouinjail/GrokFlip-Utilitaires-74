# GrokFlip • Utilitaires Haute-Savoie (74)

Agrégateur de deals utilitaires/fourgonnettes + scraping automatique Selenium + analyse investisseur cash avec Grok.

## Installation (3 minutes)

```bash
cd /chemin/vers/le/dossier
git clone https://github.com/tigrouinjail/GrokFlip-Utilitaires-74.git
cd GrokFlip-Utilitaires-74
pip install -r requirements.txt
python grokflip_utilitaires_74.py
```

Puis ouvre **http://127.0.0.1:5000** dans ton navigateur.
